## ✔️구현한 기능목록 
 - 자판기가 보유하고 있는 금액(잔돈)을 무작위로 생성한다.
 - 최소 개수의 동전으로 잔돈을 돌려준다. 
 - 사용자의 남은 금액이 상품금액보다 적으면 잔돈을 돌려준다. 
 - 모든 상품이 소진되면 잔돈을 돌려준다. 
 - 자판기에 상품명, 가격, 수량을 추가한다
